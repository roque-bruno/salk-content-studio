# Checklist de Compliance Regulatório — Squad Content Production

## Uso
Checklist exclusivo de @compliance-guardian (Shield). GATE BLOQUEANTE.
Executar para TODA peça que contenha claims técnicos, lançamentos ou cases.

---

## 1. Verificação de Claims

- [ ] TODOS os claims técnicos possuem ID do banco aprovado (ET-XX, KR-XX, CMP-XX)?
- [ ] Nenhum claim foi inventado, extrapolado ou exagerado?
- [ ] Claims comparativos incluem nota de fonte (*Baseado em dados públicos)?
- [ ] Claims comparativos NÃO citam concorrentes por nome?
- [ ] Dados numéricos conferem com manuais ANVISA oficiais?

## 2. Termos Proibidos

- [ ] Sem superlativos absolutos ("o melhor", "o mais moderno", "inigualável")?
- [ ] Sem promessas de resultado ("garante", "elimina", "cura")?
- [ ] Sem urgência falsa ("últimas unidades", "só hoje")?
- [ ] Sem comparação depreciativa ("melhor que [marca]")?
- [ ] Sem trivialização de saúde (emojis de caveira, seringa, etc.)?

## 3. Regulações Específicas

### ANVISA RDC 96/2008
- [ ] Propaganda é verdadeira (Art. 4º)?
- [ ] Sem falsa atribuição de propriedades (Art. 5º)?
- [ ] Sem sugestão de ausência de riscos (Art. 6º)?
- [ ] Sem expressões proibidas tipo "sem contra-indicação" (Art. 7º)?
- [ ] Não induz ao uso desnecessário (Art. 8º)?
- [ ] Número de registro ANVISA consta ou é citável (Art. 9º)?
- [ ] Comparações com dados científicos e fonte identificada (Art. 10º)?

### ANVISA RDC 751/2022
- [ ] Claims referenciam apenas dados registrados na ANVISA?
- [ ] Classe de risco do dispositivo respeitada na comunicação?

### CFM 2.336/2023
- [ ] Sem garantia de resultados de procedimentos?
- [ ] Sem sensacionalismo?
- [ ] Sem indução a autodiagnóstico?
- [ ] Imagens clínicas (se houver) com consentimento expresso?

## 4. Compliance Visual

- [ ] Produto é foto REAL (PNG original, não IA)?
- [ ] Aparência do produto NÃO foi alterada por IA (cor, formato, tamanho)?
- [ ] NÃO há pessoas geradas por IA?
- [ ] NÃO há cenas clínicas geradas por IA (cirurgias, pacientes)?
- [ ] Cenário de fundo (se IA) é clean/showroom, sem contexto clínico?
- [ ] Certificações exibidas são válidas e atuais?

## 5. Disclaimers

- [ ] Claims comparativos com nota de fonte?
- [ ] Registro ANVISA referenciável?
- [ ] "Equipamento para uso profissional" (quando aplicável)?

---

## Veredito

| Veredito | Critério | Ação |
|----------|---------|------|
| ✅ **APROVADO** | Todos os itens OK | Liberar para publicação |
| ⚠️ **RESSALVA** | Risco médio sem violação direta | Aprovar + feedback |
| ❌ **BLOQUEADO** | Qualquer violação regulatória | Devolver + fundamentação |

### Formato do Veredito
```
[VEREDITO]: ✅/⚠️/❌
Peça: [ID]
Claims validados: [IDs]
Regulação aplicável: [citação]
Observações: [se houver]
Ação requerida: [se bloqueado]
```
