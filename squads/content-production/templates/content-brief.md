# Content Brief Template — Squad Content Production

## Uso
Template padrão para briefing de cada peça de conteúdo. Gerado por @content-strategist (Atlas).
Todos os campos são obrigatórios para peças com claims técnicos. Peças genéricas podem omitir Claims.

---

```
=== CONTENT BRIEF ===

ID: [MARCA]-[ANO]-[SEMANA]-[NUMERO]
Ex: SALK-2026-W15-001

--- INFORMAÇÕES BÁSICAS ---
Marca: [Salk Medical / Mendel Medical / Manager Grupo / Dayho]
Produto/Serviço: [LEV / KRATUS / OSTUS / KRONUS / COMMAND / Institucional]
Plataforma(s): [Instagram / LinkedIn / YouTube / Facebook]
Formato: [Post / Carousel / Reel / Story / Vídeo / PDF]
Pilar de Conteúdo: [Produto / Educacional / Cases / Bastidores / Thought Leadership]
Objetivo de Funil: [Awareness / Consideração / Avaliação / Decisão / Licitação]
Buyer Persona Alvo: [Compras / Eng. Clínica / Equipe Médica / Administradores / Geral]

--- BRIEFING CRIATIVO ---
Mensagem principal: [1 frase clara]
Framework de copy: [PAS / AIDA / BAB / StoryBrand / SPIN]
Tom de voz: [Consultivo / Técnico / Acolhedor / Industrial]
CTA: [Ação desejada — ex: "Agende demonstração técnica"]
Hashtags: [Selecionar do banco padronizado — máx 11 IG, 5 LI]

--- CONTEÚDO ---
Headline/Hook: [Frase de abertura — máx 125 chars para IG]
Corpo: [Texto principal ou script com timecodes]
Claims utilizados: [IDs do banco: LEV-01, KR-03, CMP-01, etc.]
Fontes/Referências: [Manuais ANVISA, documentação oficial]

--- ESPECIFICAÇÕES TÉCNICAS ---
Dimensões: [1080x1350 / 1080x1080 / 1080x1920 / 1200x627 / 1280x720]
Template Canva: [Nome/ID do template]
Assets necessários: [Fotos PNG do produto, cenários IA, vídeos]
Duração (vídeo): [Xs — ex: 30s para Reel, 5min para demo]

--- ATOMIZAÇÃO ---
É peça-mãe? [Sim / Não]
Derivados planejados: [Lista de derivados cross-platform]
Plataformas derivadas: [IG Feed, IG Reel, IG Story, LI, YT Short, FB]

--- REVISÃO ---
Nível de revisão: [Camadas 1+2 / 1+2+3 / 1+2+3+4]
Data limite de aprovação: [DD/MM/AAAA]
Status: [Rascunho / Em produção / Em revisão / Aprovado / Publicado]

--- DISTRIBUIÇÃO ---
Data de publicação: [DD/MM/AAAA]
Horário: [HH:MM]
UTM: [?utm_source=XX&utm_medium=XX&utm_campaign=XX&utm_content=XX]
Agendado em: [Buffer / Later / Meta Suite / Nativo]
```
