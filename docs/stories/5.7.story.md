# Story 5.7 — Relatorio Semanal PDF

**Status:** Done
**Epic:** 5 — Escala + Publicacao

## Objetivo

Implementar WeeklyReport que gera relatorios em JSON e HTML com resumo de producao, metricas de engajamento, top posts, custos e insights, com template HTML profissional pronto para impressao em PDF.

## Acceptance Criteria

- [x] Classe WeeklyReport implementada com geracao de relatorios JSON e HTML
- [x] Resumo de producao semanal (pecas criadas, publicadas, pendentes)
- [x] Metricas de engajamento agregadas por plataforma
- [x] Ranking de top posts da semana
- [x] Resumo de custos (tokens LLM, API calls)
- [x] Insights e recomendacoes automaticas
- [x] Template HTML profissional pronto para impressao em PDF via browser
- [x] Testes unitarios cobrindo geracao de relatorios

## Arquivos

- `publishers/report_pdf.py` — WeeklyReport class com geracao JSON e HTML

## API Endpoints

- `GET /api/report/weekly` — Relatorio semanal em JSON
- `GET /api/report/weekly/html` — Relatorio semanal em HTML para impressao PDF
