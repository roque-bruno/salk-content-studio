### 3. Teste E2E Manual

1. Abrir https://studio.salkmedical.com
2. Criar nova peca (todos os campos)
3. Gerar copy individual
4. Gerar copy batch
5. Gerar prompt individual
6. Gerar prompt batch
7. Gerar imagem individual (com e sem produto)
