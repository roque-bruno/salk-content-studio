# Story 5.1 — Publisher Worker Instagram

**Status:** Done
**Epic:** 5 — Escala + Publicacao

## Objetivo

Implementar InstagramPublisher com integracao ao Graph API para publicacao de conteudo no Instagram, suportando imagem unica, carousel, reels e stories, com modo preview quando nao ha access token.

## Acceptance Criteria

- [x] Classe InstagramPublisher implementada com integracao Graph API
- [x] Suporte a publicacao de imagem unica
- [x] Suporte a publicacao de carousel (multiplas imagens)
- [x] Suporte a publicacao de reels (video curto)
- [x] Suporte a publicacao de stories
- [x] Modo preview funcional quando access token nao configurado
- [x] Herda de PublisherBase com interface padronizada
- [x] Tratamento de erros e retries para falhas de API
- [x] Testes unitarios cobrindo todos os formatos

## Arquivos

- `publishers/base.py` — PublisherBase abstract class, PublishResult e PostMetrics dataclasses usadas por todos os publishers
- `publishers/instagram.py` — InstagramPublisher class com Graph API integration

## API Endpoints

- `POST /api/publish/instagram` — Publicar conteudo no Instagram
