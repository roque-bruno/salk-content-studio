# Story 4.7 — CRON Semanal (Agendamento)

**Epico:** 4 — Automacao + Inteligencia
**Status:** Planned
**Executor:** @devops (Gage)

---

## Descricao

Agendamento semanal automatizado para geracao de conteudo. Integra o pipeline completo (briefing → copy → prompt → geracao → disaster check → atomizacao) em um CRON que roda semanalmente, produzindo o batch de conteudo planejado no calendario editorial.

## Acceptance Criteria

- [ ] CRON semanal configurado para executar pipeline completo
- [ ] Leitura automatica do calendario editorial da semana
- [ ] Execucao sequencial das etapas do pipeline
- [ ] Relatorio de execucao com status de cada peca gerada
- [ ] Notificacao de falhas para revisao humana
- [ ] Retry automatico em falhas transientes
- [ ] Deploy via GitHub Actions ou scheduler equivalente

## Tasks

1. Definir infraestrutura de agendamento (GitHub Actions / VPS)
2. Implementar script de orquestracao do pipeline semanal
3. Integrar leitura do calendario editorial
4. Implementar relatorio de execucao
5. Configurar notificacoes de falha
6. Implementar retry em falhas transientes
7. Configurar deploy e monitoramento

## Dependencias

- Stories 4.1-4.6 (pipeline completo implementado)
- Infraestrutura de deploy (@devops)
