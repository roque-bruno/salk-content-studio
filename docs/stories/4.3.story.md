# Story 4.3 — Persona Clones (Teste de Copy)

**Epico:** 4 — Automacao + Inteligencia
**Status:** Done
**Executor:** @dev (Dex)

## Descricao

4 simulacoes de buyer personas para testar efetividade de copy antes da publicacao. Cada PersonaClone avalia o copy sob a perspectiva do publico-alvo da marca, gerando score de relevancia, clareza, persuasao e feedback qualitativo.

## Acceptance Criteria

- [x] Classe PersonaClone implementada com metodo evaluate()
- [x] 4 personas configuradas alinhadas com as marcas do grupo
- [x] evaluate() retorna score de relevancia, clareza e persuasao
- [x] evaluate() retorna feedback qualitativo via LLM
- [x] Endpoint para listar personas disponiveis
- [x] Integracao com output dos copywriters (Story 4.2)
- [x] Testes unitarios cobrindo cada persona

## Tasks

1. Implementar classe PersonaClone em `automation/copywriter.py`
2. Configurar 4 buyer personas alinhadas com as marcas
3. Implementar evaluate() com scoring multi-dimensional
4. Implementar feedback qualitativo via LLM
5. Implementar endpoints da API
6. Escrever testes unitarios

## API Endpoints

- `POST /api/automation/persona/evaluate` — Avaliar copy com persona
- `GET /api/automation/persona/list` — Listar personas disponiveis
