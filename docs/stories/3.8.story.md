## Descricao

Sistema de registro de decisoes e resultados do pipeline de producao para self-learning. Registra cada decisao tomada (modelo escolhido, prompt usado, parametros), o resultado obtido (qualidade, feedback humano, metricas) e estatisticas agregadas para que o sistema aprenda com o historico e melhore decisoes futuras.
