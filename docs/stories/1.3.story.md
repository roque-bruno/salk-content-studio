# Story 1.3 — Calibrar NB2 com hero shots individuais (LEV + KRATUS)

**Epico:** 1 — Calibracao & Primeiro Batch Real
**Status:** InProgress
**Executor:** Apex + Usuario
**PRD:** docs/prd/PRD-content-production.md v2.0
**Depende de:** Story 1.2

---

## Descricao

Produzir hero shots NB2 individuais para LEV e KRATUS ate atingir aprovacao do usuario. Usar arquitetura de 8 dimensoes validada. Para LEV: tecnica Dramatic Studio com fundo escuro e reflexo. Iterar prompts ate qualidade de advertising profissional.

## Acceptance Criteria

- [ ] Hero shot LEV aprovado pelo usuario (NB2, Dramatic Studio, 8 dimensoes)
- [ ] Hero shot KRATUS aprovado pelo usuario (NB2, tecnica adequada ao produto)
- [ ] Prompts vencedores documentados como referencia para Apex
- [ ] Composicao Canva aplicada ao hero shot (logo, headline, spec line, gradientes)
- [ ] Resultado final com qualidade de advertising profissional
- [ ] Processo documentado: quantas iteracoes, o que funcionou, o que nao funcionou

## Tasks

1. Apex gera prompt NB2 para LEV hero shot usando Dramatic Studio + 8 dimensoes
2. Usuario executa no Google AI Studio com upload do PNG LEV
3. Avaliar resultado → se reprovado, Apex itera prompt (V2, V3...)
4. Repetir ate aprovacao (max 5 iteracoes por conceito, trocar conceito se necessario)
5. Repetir processo para KRATUS
6. Aplicar composicao Canva (margens 60px, logo 180-200px, gradientes)
7. Documentar prompts vencedores e processo

## Regras NB2 (PRD v2.0 Secao 6)

- Descrever APENAS cenario, NUNCA o produto
- Negative prompt obrigatorio (sem texto, logo, equipamentos medicos)
- LEV: luz focada para baixo, NUNCA dispersa/lateral
- Produto como 60-70% do frame
- Camera e lente reais nomeadas
- Instrucao explicita: "Place this EXACT uploaded product, do NOT modify"
