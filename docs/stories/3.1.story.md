## Descricao

Cliente Python para a API Kling 3.0 Pro com suporte a Image-to-Video (I2V), retry com backoff exponencial, polling de status de tarefas e download automatico do video gerado. Encapsula autenticacao, envio de imagem, configuracao de parametros de geracao e tratamento de erros da API.

## Acceptance Criteria

- [x] Cliente Python implementado com classe KlingClient
- [x] Autenticacao via API key com header configuravel
- [x] Endpoint Image-to-Video (I2V) funcional com upload de imagem
- [x] Retry automatico com backoff exponencial em falhas transientes
- [x] Polling de status da task ate conclusao ou timeout
- [x] Download automatico do video gerado para diretorio de saida
- [x] Tratamento de erros com mensagens claras (rate limit, auth, timeout)
- [x] Testes unitarios cobrindo cenarios de sucesso e falha
