## Descricao

Integracao com OpenRouter para roteamento inteligente de chamadas LLM entre multiplos modelos. Suporta 4 modelos com task routing automatico baseado no tipo de tarefa (copy, analise, planejamento, revisao), fallback entre modelos e tracking de custos por chamada.

## Acceptance Criteria

- [x] Cliente Python para API OpenRouter implementado
- [x] Suporte a 4 modelos com configuracao de prioridade
- [x] Task routing automatico baseado no tipo de tarefa
- [x] Fallback automatico para modelo alternativo em caso de falha
- [x] Tracking de custo por chamada (tokens input/output + preco)
- [x] Rate limiting e retry com backoff
- [x] Logging estruturado de todas as chamadas com metricas
- [x] Testes unitarios cobrindo routing e fallback
