## Descricao

Arquitetura de prompts NB2 com 8 dimensoes para geracao de imagens. Inclui regras especificas por produto (LEV: luz concentrada no campo, nunca dispersao lateral), negative prompts obrigatorios (sem equipamentos concorrentes, sem texto/logo em IA) e modo criativo com variacao de conceito.

## Acceptance Criteria

- [x] Arquitetura de 8 dimensoes implementada para prompts NB2
- [x] Regras especificas por produto (LEV light direction, etc)
- [x] Negative prompts obrigatorios em todos os prompts gerados
- [x] Modo criativo com variacao de conceito visual
- [x] Endpoint para listar dimensoes e suas configuracoes
- [x] Produto nunca descrito no prompt (apenas cenario, produto via upload)
- [x] Sem equipamentos medicos concorrentes nos prompts
- [x] Testes unitarios cobrindo regras de cada produto

## API Endpoints

- `POST /api/automation/prompt/generate` — Gerar prompt NB2
- `POST /api/automation/prompt/generate-creative` — Gerar prompt criativo com variacao
- `GET /api/automation/prompt/dimensions` — Listar dimensoes e configuracoes
