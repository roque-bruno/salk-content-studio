## Descricao

Fazer deploy do frontend estatico no subdominio `studio.salk.com` via cPanel UAPI. Configurar DNS, criar subdominio, upload dos arquivos HTML/CSS/JS. O frontend deve estar acessivel publicamente em https://studio.salk.com.

## Acceptance Criteria

- [ ] Subdominio `studio.salk.com` criado no cPanel
- [ ] Certificado SSL configurado (Let's Encrypt via cPanel)
- [ ] Arquivos do frontend deployed em `public_html/studio/`
- [ ] `config.js` configurado com URL da API de producao
- [ ] Site acessivel em https://studio.salk.com
- [ ] Redirecionamento HTTP → HTTPS ativo
- [ ] Tempo de carregamento < 3 segundos

## Tasks

1. Criar subdominio `studio.salk.com` via cPanel
2. Configurar SSL via cPanel AutoSSL
3. Upload dos arquivos do frontend
4. Configurar `config.js` com API_BASE_URL de producao
5. Configurar `.htaccess` para SPA routing e cache
6. Testar acesso publico
7. Verificar HTTPS redirect
