**Epico:** 4 — Automacao + Inteligencia
**Status:** Done
**Executor:** @dev (Dex)

## Descricao

Transforma conteudo master em derivados com logica semantica. Suporta 4 tipos de master (post carrossel, video curto, artigo, hero shot) e 7 formatos de plataforma (Instagram feed/stories/reels, LinkedIn, Facebook, TikTok, WhatsApp). Usa LLM para adaptacao inteligente com fallback para truncamento basico.

## Acceptance Criteria

- [x] 4 tipos de master suportados (carrossel, video curto, artigo, hero shot)
- [x] 7 formatos de plataforma configurados
- [x] Adaptacao semantica via LLM (tom, formato, CTA por plataforma)
- [x] Fallback para truncamento basico quando LLM indisponivel
- [x] Sugestao automatica de derivados a partir de master
- [x] Endpoint para listar plataformas e seus formatos
- [x] Ratio de atomizacao configuravel (1:5-8 por master)
- [x] Testes unitarios cobrindo cada tipo de master e plataforma

## API Endpoints

- `POST /api/automation/atomize` — Atomizar conteudo master
- `POST /api/automation/atomize/suggest` — Sugerir derivados para master
- `GET /api/automation/atomize/platforms` — Listar plataformas e formatos
