## Descricao

Deploy da API FastAPI no Render.com (free tier). Configurar auto-deploy via GitHub, variaveis de ambiente, e health check. A API deve estar acessivel publicamente para o frontend em studio.salk.com consumir.

## Acceptance Criteria

- [ ] Servico Web criado no Render (Python, free tier)
- [ ] Auto-deploy conectado ao repositorio GitHub
- [ ] Variaveis de ambiente configuradas (GOOGLE_API_KEY, ALLOWED_ORIGINS)
- [ ] Health check `/api/health` respondendo com status 200
- [ ] CORS aceita requests de studio.salk.com
- [ ] `requirements.txt` ou `pyproject.toml` configurado para Render
- [ ] Render.yaml ou Procfile criado
- [ ] API estavel com auto-restart em caso de crash

## Tasks

1. Criar `render.yaml` com configuracao do servico
2. Criar `Procfile` alternativo
3. Configurar variaveis de ambiente no painel Render
4. Conectar repo GitHub ao Render
5. Testar deploy e health check
6. Validar CORS com frontend
