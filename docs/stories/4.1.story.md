## Descricao

Classe AutoBriefing que gera briefings automaticos a partir de slots do calendario editorial via LLM. Atlas analisa o slot (marca, produto, tipo de conteudo, data) e produz um briefing completo com objetivo, tom, claims sugeridos e diretrizes visuais.

## Acceptance Criteria

- [x] Classe AutoBriefing implementada com geracao de briefings via LLM
- [x] Integracao com calendario editorial para leitura de slots
- [x] Geracao individual de briefing por slot
- [x] Geracao em batch de multiplos briefings
- [x] Briefing inclui objetivo, tom, claims, diretrizes visuais e restricoes
- [x] Respeita regras de marca (Mendel=tecnico, Salk=comercial)
- [x] Tratamento de erros e fallback quando LLM indisponivel
- [x] Testes unitarios cobrindo cenarios de sucesso e falha
