# Story 5.8 — Batch Approval Workflow

**Status:** Done
**Epic:** 5 — Escala + Publicacao

## Objetivo

Implementar BatchApproval para que a gestora de marketing aprove ou rejeite pecas em batch, com integracao de compliance check e criacao automatica de review na aprovacao.

## Acceptance Criteria

- [x] Classe BatchApproval implementada para aprovacao/rejeicao em batch
- [x] Listagem de pecas pendentes de aprovacao
- [x] Aprovacao individual e em batch
- [x] Rejeicao com motivo obrigatorio
- [x] Integracao com compliance check antes da aprovacao
- [x] Criacao automatica de review na aprovacao
- [x] Resumo de aprovacoes/rejeicoes com estatisticas
- [x] Testes unitarios cobrindo fluxo completo

## Arquivos

- `publishers/batch_approval.py` — BatchApproval class com workflow de aprovacao

## API Endpoints

- `GET /api/approval/pending` — Listar pecas pendentes de aprovacao
- `POST /api/approval/approve` — Aprovar pecas (individual ou batch)
- `POST /api/approval/reject` — Rejeitar pecas com motivo
- `GET /api/approval/summary` — Resumo de aprovacoes e rejeicoes
