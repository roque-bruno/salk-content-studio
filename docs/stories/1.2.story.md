## Descricao

Preparar todos os assets visuais necessarios para producao NB2: identificar PNGs com melhor qualidade para cada produto, verificar quais ja tem fundo transparente, confirmar paleta de cores HEX por marca, e criar templates Canva base.

## Acceptance Criteria

- [x] Inventario de PNGs: listar melhores imagens de LEV e KRATUS para NB2 (minimo 3 por produto)
- [x] PNGs com fundo transparente identificados ou criados (minimo 2 LEV + 2 KRATUS)
- [x] Paleta de cores HEX confirmada para Salk Medical e Mendel Medical
- [x] Template Canva base criado: feed Instagram 1080x1350 com margens 60px, zona de logo, gradientes
- [x] Template Canva base criado: stories 1080x1920
- [x] Documento de assets prontos criado: lista de PNGs validados com paths
