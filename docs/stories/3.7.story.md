## Descricao

Dashboard de monitoramento de gastos com APIs em tempo real. Armazena custos por provider (OpenRouter, Kling, ElevenLabs, Gemini) em SQLite, calcula totais por periodo, emite alertas quando limites configurados sao atingidos e expoe metricas para o frontend.

- [x] Alertas configuraveis por limite de gasto (warning e critical)
- [x] Endpoint API para consulta de gastos em tempo real
- [x] Dashboard summary com breakdown por provider
- [x] Testes unitarios com banco SQLite in-memory
