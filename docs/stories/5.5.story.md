# Story 5.5 — Metrics Collector Automatico

**Status:** Done
**Epic:** 5 — Escala + Publicacao

## Objetivo

Implementar MetricsCollector que coleta metricas de todas as plataformas 48h apos publicacao, com suporte a coleta individual e em batch.

## Acceptance Criteria

- [x] Classe MetricsCollector implementada com coleta de todas as plataformas
- [x] Coleta automatica 48h apos publicacao
- [x] Suporte a coleta individual por plataforma e post
- [x] Suporte a coleta em batch de multiplos posts
- [x] Armazenamento de metricas com historico
- [x] Tratamento de erros e retries para falhas de API
- [x] Testes unitarios cobrindo cenarios de coleta

## Arquivos

- `publishers/base.py` — PublisherBase abstract class, PublishResult e PostMetrics dataclasses
- `publishers/metrics_collector.py` — MetricsCollector class com coleta multi-plataforma

## API Endpoints

- `POST /api/metrics/collect` — Coletar metricas de todos os posts pendentes
- `POST /api/metrics/collect/{platform}/{post_id}` — Coletar metricas de um post especifico
- `GET /api/metrics/collector-status` — Status do collector e posts pendentes
