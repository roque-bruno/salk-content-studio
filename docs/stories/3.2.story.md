## Descricao

Pipeline completo que conecta a geracao de hero shots NB2 ao Kling 3.0 Pro para animacao automatica. Recebe uma imagem hero gerada pelo pipeline de conteudo, envia para o Kling via I2V, aguarda processamento e entrega o video de saida pronto para montagem. Requer integracao entre os modulos existentes de geracao de imagem e o cliente Kling.

## Acceptance Criteria

- [ ] Pipeline recebe path de imagem NB2 hero e gera video animado
- [ ] Integracao com KlingClient (Story 3.1) para envio I2V
- [ ] Configuracao de parametros de animacao (duracao, movimento, estilo)
- [ ] Fila de processamento para multiplas imagens
- [ ] Callback ou polling para notificar conclusao
- [ ] Armazenamento organizado dos videos gerados por campanha
- [ ] Fallback para erro de geracao com retry e notificacao
- [ ] Logging completo do pipeline com tempos de execucao

## Tasks

1. Definir interface do pipeline I2V
2. Integrar com modulo de geracao NB2
3. Conectar com KlingClient para envio e polling
4. Implementar fila de processamento
5. Adicionar organizacao de output por campanha
6. Implementar fallback e notificacao de erros
7. Adicionar logging e metricas de tempo
8. Testes de integracao
